## About

Forward Together reimagines cancer support. No clinical questionnaires. No waiting rooms. Just meaningful connection that meets people exactly where they are.

### Features

- **Bob** - AI Survivor Guide for those facing a diagnosis
- **Miri** - AI Caregiver Guide for those supporting a loved one
- **Peer Matching** - Connect with people who've walked your path
- **Collective Wisdom** - Learn from thousands of journeys
- **24/7 Availability** - Always here when you need us

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Fraunces & DM Sans fonts

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR-USERNAME/forward-together.git
cd forward-together

# Install dependencies
npm install

# Start development server
npm run dev
```

The site will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

Built files will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```
